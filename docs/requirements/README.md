# 요구사항 위키

- 문서 목적: 요구사항 문서를 `CRS`와 `SRS` 중심으로 탐색할 수 있는 진입점을 제공한다.
- 범위: `docs/requirements/` 아래 문서
- 대상 독자: 기획자, 개발자, 아키텍트, 운영자, 테스트 담당자
- 상태: draft
- 최종 수정일: 2026-04-04
- 관련 문서: `docs/README.md`, `docs/overview/README.md`

## 문서 구성 원칙

- `CRS`는 과제 컨셉과 상위 수준 요구를 정리한다.
- `SRS`는 `CRS`를 세분화한 상세 요구사항을 정리한다.
- 컨셉 변경이 발생하면 `CRS`와 `SRS`를 함께 검토한다.
