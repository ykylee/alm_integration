# 의사결정 위키

- 문서 목적: 설계 의사결정과 ADR 문서를 저장하고 탐색할 수 있는 카테고리 진입점을 제공한다.
- 범위: `docs/decisions/` 아래 의사결정 기록, ADR, 선택 근거, 대안 비교 문서
- 대상 독자: 개발자, 아키텍트, 운영자, 리뷰어, 문서 작성자
- 상태: draft
- 최종 수정일: 2026-04-17
- 관련 문서: `docs/README.md`, `docs/architecture/README.md`, `docs/operations/document_sync_policy.md`

## 이 카테고리에서 다루는 내용

- 주요 설계 선택의 배경
- 고려한 대안과 제외 이유
- 결정 시점의 제약과 가정
- 후속 영향과 재검토 조건

## 현재 상태

- 아직 등록된 ADR 문서는 없다.
- 다만 문서 체계 상 `decisions/` 카테고리는 이미 기준 구조에 포함돼 있으므로, 이후 주요 설계 선택은 이 경로에 누적한다.

## 권장 작성 기준

- 문서 목적
- 범위
- 대상 독자
- 상태
- 최종 수정일
- 관련 문서
- 결정 배경
- 선택한 안
- 대안 비교
- 영향 범위
- 재검토 조건
